

# Frieren-MD

A simple *lightweight* whatsapp BOT using *Baileys Multi-device*


## Installation

Clone the repo

```bash
git clone https://github.com/xct007/frieren-md
```

Install Packages
```bash
npm install --save
```
or using yarn
```bash
yarn install
```    
## Run

Simple in terminal

```bash
npm run start
```
